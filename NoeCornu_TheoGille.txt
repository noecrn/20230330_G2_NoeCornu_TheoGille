# Jeu de tir en réalité augmentée  

Ce projet est un jeu de tir en réalité augmentée, où le joueur doit déplacer son téléphone pour trouver et détruire des cibles en mouvement. Le jeu dispose de deux modes différents : un mode chronométré, où le joueur doit détruire un certain nombre de cibles dans un temps imparti, et un mode sandbox sans limite de temps.

## Fonctionnalités  
* Les joueurs peuvent choisir entre deux modes de jeu : chronométré et sandbox
* Dans le mode chronométré, les joueurs doivent détruire un certain nombre de cibles dans un temps imparti pour gagner
* Dans le mode sandbox, les joueurs peuvent détruire des cibles à l'infini sans limite de temps
* La scène de jeu contient un timer pour afficher le temps restant dans le mode chronométré
* Un compteur affiche le nombre de cibles détruites
* Un son de tir de fusil est joué lorsqu'une cible est détruite
* Un fusil en 3D est fixé à la caméra pour simuler un jeu de tir à la première personne en AR
* Des particules apparaissent lors de la destruction d'une cible

## Instructions d'installation  
1. Téléchargez le projet depuis GitHub
2. Ouvrez le projet dans Unity.
3. Branchez votre téléphone à votre ordinateur en utilisant un câble USB.
4. Activez le mode développeur sur votre téléphone.
5. Dans Unity, accédez à File > Build Settings, et sélectionnez votre téléphone dans la liste des périphériques disponibles.
6. Cliquez sur Build and Run pour compiler le projet et l'installer sur votre téléphone.

## Configurations requises  
* Unity 2019.4 ou version ultérieure
* Un téléphone Android compatible avec la réalité augmentée
